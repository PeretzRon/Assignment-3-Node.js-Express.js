# Assignemnt 3 - Node.js - Express.js (Udemy)

- Create a npm project and install Express.js.
-  Create an Express.js app which serves two HTML files for '/' and '/users'.
- Add some static (.js or .css) files to your project that should be required by at least one of your HTML files

## How to use

1. Run "npm install" in the extracted folder
2. Run "npm start" to view the project
